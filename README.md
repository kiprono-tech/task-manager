# Vincent Kiprono

## Project setup

### Client

- install dependencies
```bash
    npm install
```
### Server

- install dependencies
```bash
    npm install
```

- build project
```bash
    npm run build
```

- run web app
```bash
    npm run start
```

- seed databse
```bash
    npm run seed
```

- run in development
```bash
    npm run start:dev
```

- run in production
```bash
  npm run start
```
